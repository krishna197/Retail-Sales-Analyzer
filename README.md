# 🛍️ Retail Sales Analyzer

A Python-based retail sales analysis project that processes sales data, generates business insights, and visualizes sales trends using **Pandas** and **Matplotlib**.

## 📌 Project Overview

The Retail Sales Analyzer helps analyze retail sales data by performing data processing, calculating important business metrics, and creating visualizations for better decision-making.

The project includes:

- Data loading and preprocessing
- Sales analysis
- Product performance analysis
- Daily sales calculation
- Data visualization

---

## 🚀 Features

- 📊 Calculate total sales for each product
- 🏆 Identify the best-selling product
- 📅 Calculate average daily sales
- 📈 Generate sales trend charts
- 🧹 Clean and process sales data automatically

---

## 🛠️ Technologies Used

- Python 3.x
- Pandas
- Matplotlib

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/krishna197/Retail-Sales-Analyzer.git
```

Move into the project folder:

```bash
cd Retail-Sales-Analyzer
```

Install dependencies:

```bash
pip install pandas matplotlib
```

or

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Project

Open the notebook:

```bash
jupyter notebook
```

or

```bash
Retail_Sales_Analyzer.ipynb
```

Run all cells to perform the analysis.
